## Calendar Facts
Randomly generate cool calendar facts.
See [this XKCD](http://xkcd.com/1930).

### Running the script
You need to have python installed.

1. Clone this project
1. Using a command line program, go to the cloned directory
1. run `python calendar_facts.py`
